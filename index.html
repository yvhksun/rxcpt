会员注册

  1. 进入融信城首页、点击“会员登录”按钮
  2. 点击免费注册
  3. 输入手机号点击获取验证码，系统会将验证码发送到你的手机，输入验证码后，设置8位密码点击确认注册成为融信城会员
  4. 注册时的手机号为融信城会员的用户名

企业注册

  1. 依次输入账户邮箱、密码、确认密码、公司全称、手机号码点 　击获取验证码，系统会发送验证码至您手机，输入验证码后点击注册
  2. 提交后会发送验证至邮箱
  3. 邮箱验证成功
  4. 完成注册

商户认证

  1. 填写公司信息，点击下一步（红色星号必填）
  2. 填写负责人信息，并勾选同意条款
  3. 认证提交融信城审核
  4. 融信城审核通过，会发送短信给商户预留手机，并同时抄送至 　您的邮箱，商户重新登录后即为认证成功

一键查询

空蓝内输入您需要安检查询的金融公司简称或两个字以上的关 键字，点击快速查询即可

快速提问

回复解答

公开浏览
